# Regional Licenses Module

## Purpose
Management of regional tourism licenses (e.g., GVA in Comunitat Valenciana).

## Public Interface
- **RegionalLicenseTypes**: TypeScript types for regional licenses
- **RegionalLicenseService**: License lookup and validation

## Dependencies
- sharedkernel
- connectors

## Usage
```typescript
import { regionalLicenseService } from "@/regionallicenses/RegionalLicenseService"
```

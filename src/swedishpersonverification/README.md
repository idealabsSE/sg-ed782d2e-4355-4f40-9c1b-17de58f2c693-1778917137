# Swedish Person Verification Module

## Purpose
Identity verification for Swedish nationals using Swedish BankID and providers.

## Public Interface
- **SwedishVerificationService**: Orchestrate Swedish identity verification
- **SwedishVerificationTypes**: TypeScript types for Swedish verification

## Dependencies
- sharedkernel
- auth
- documents

## Usage
```typescript
import { swedishVerificationService } from "@/swedishpersonverification/SwedishVerificationService"
```

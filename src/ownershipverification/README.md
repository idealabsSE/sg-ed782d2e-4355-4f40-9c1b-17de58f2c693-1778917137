# Ownership Verification Module

## Purpose
Verify property ownership through Nota Simple document analysis.

## Public Interface
- **OwnershipService**: Ownership verification workflow
- **OwnershipTypes**: TypeScript types for ownership verification

## Dependencies
- sharedkernel
- auth
- documents
- properties

## Usage
```typescript
import { ownershipService } from "@/ownershipverification/OwnershipService"
```

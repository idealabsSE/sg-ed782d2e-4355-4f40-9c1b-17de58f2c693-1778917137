# Spanish Person Verification Module

## Purpose
Identity verification for Spanish nationals using Spanish identity documents.

## Public Interface
- **SpanishVerificationService**: Orchestrate Spanish identity verification
- **SpanishVerificationTypes**: TypeScript types for Spanish verification

## Dependencies
- sharedkernel
- auth
- documents

## Usage
```typescript
import { spanishVerificationService } from "@/spanishpersonverification/SpanishVerificationService"
```

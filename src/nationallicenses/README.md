# National Licenses Module

## Purpose
Management of national-level property licenses and registrations.

## Public Interface
- **NationalLicenseTypes**: TypeScript types for national licenses
- **NationalLicenseService**: License lookup and validation

## Dependencies
- sharedkernel
- connectors

## Usage
```typescript
import { nationalLicenseService } from "@/nationallicenses/NationalLicenseService"
```

# Security and Audit Module

## Purpose
Audit logging, security monitoring, and compliance tracking.

## Public Interface
- **AuditService**: Audit log operations
- **SecurityService**: Security tests and monitoring
- **AuditTypes**: TypeScript types for audit entities

## Dependencies
- sharedkernel
- auth

## Usage
```typescript
import { auditService } from "@/securityandaudit/AuditService"
```
